Test Environment
\begin{itemize}
\item Single Host
\item CPU: Intel(R) Core(TM) i5-3570K CPU @ 3.40GHz
\item RAM: 12GB @ 1600
\item network size: 30 Nodes
\item storage mode: MEMORY, to reduce effects of (disk) IO delay
\item replication factor: 5, means actual 6 copies of each artifact stored in the network
\item all operations performed on randomly selected nodes
\item except for write performance data, all methods use very small envelopes with only a few bytes content
\end{itemize}
